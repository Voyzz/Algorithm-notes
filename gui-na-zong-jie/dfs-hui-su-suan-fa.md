# DFS/回溯算法

## 思想

* 解决一个回溯问题，实际上就是一个【决策树的遍历】过程
  * **路径**：也就是已经做出的选择。
  * **选择列表**：也就是你当前可以做的选择。
  * **结束条件**：也就是到达决策树底层，无法再做选择的条件。
* 空间复杂度：O\(logn\)

## 框架

```python
result = []
// 结点
def backtrack(路径, 选择列表):
    if 满足结束条件:
        result.add(路径)
        return

    // 子节点遍历
    for 选择 in 选择列表:
        做选择
        backtrack(路径, 选择列表)
        撤销选择
```
